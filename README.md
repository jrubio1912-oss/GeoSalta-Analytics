# 🗺️ GeoSalta Analytics

**GeoSalta Analytics** es una plataforma web desarrollada en Python para el análisis territorial de la provincia de Salta mediante mapas interactivos y dashboards dinámicos.

## Objetivo

Facilitar la visualización y el análisis de indicadores geográficos utilizando cartografía oficial y datos provenientes de archivos Excel.

## Tecnologías

* Python
* Streamlit
* Folium
* GeoPandas
* Pandas
* Plotly
* Git
* GitHub
