# Roadmap - GeoSalta Analytics

## Visión del proyecto

GeoSalta Analytics tiene como objetivo convertirse en una plataforma web de inteligencia territorial para la provincia de Salta, permitiendo explorar, analizar y visualizar información geográfica, demográfica, económica y financiera mediante mapas interactivos y dashboards.

- Arquitectura modular
